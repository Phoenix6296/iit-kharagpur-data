Shell Scripting Assignment

Instructions to Run the Scripts:

1. Ensure that you have a Linux environment set up (e.g., Ubuntu). You may also use VirtualBox + Linux image if you do not have a Linux system installed.
2. Open the terminal in the directory where the shell scripts are located.
3. Run the following commands as per the instructions below.



Question 1: Basic Calculator [calculator.sh]
- Usage: bash calculator.sh 10 + 5


Question 2: Finding LCM of Numbers [lcm.sh]
- Usage: bash lcm.sh numbers_2.txt


Question 3: Random Password Generator [password.sh]
- Usage: bash password.sh 5


Question 4: Text File Analysis Script [wordcount.sh]
- Usage: bash wordcount.sh <input_file> <search_word>
- Example: bash wordcount.sh file_4.txt fifa


Question 5: API Fetch Script [api-fetch.sh]
- Usage: bash api-fetch.sh <search_label>
- Example: bash api-fetch.sh Enhancement


Question 6: To-Do List Manager Script [todo.sh]
- Usage: 
  - Add: ./todo.sh add "<task_description>" <priority>
  - Show: ./todo.sh show
  - Complete: ./todo.sh complete <task_id>

Question 7: Search Books Script [search_books.sh]
- Usage: bash search_books.sh <input_file> <search_phrase> <context_type> <number_of_words>
- Example: bash search_books.sh books_7.txt "dark night" -both 3

Question 8: Log File Analyzer [analyselog.sh]
- Usage: bash analyselog.sh <log_file> <time_window_in_minutes>
- Example: bash analyselog.sh access_8.log 15

Question 9: Alternating Case Text Processor [alternatecase.sh]
- Usage: bash alternatecase.sh <input_file> <search_word>
- Example: bash alternatecase.sh file_9.txt test

Question 10: Lexicographic Text Processor [lexiographic.sh]
- Usage: bash lexiographic.sh <input_dir> <output_dir>
- Example: bash lexiographic.sh input_dir_10 output_dir
